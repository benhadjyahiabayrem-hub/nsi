Détails Techniques de l'Architecture

Structure de Données Modulaire

Au lieu de figer le contenu dans le fichier HTML, l'application utilise une structure de données JavaScript (un tableau d'objets complexes). Cela permet une scalabilité totale de l'application.

Manipulation Dynamique du DOM

Le moteur JavaScript génère dynamiquement l'interface utilisateur, écoute en temps réel les changements d'état (événements change et click), gère une barre de progression mathématiquement proportionnelle, et applique des classes CSS dynamiques pour indiquer visuellement la justesse des réponses.

Design System & Adaptabilité

Le design repose sur une architecture CSS utilisant des variables (custom properties), Flexbox, CSS Grid pour les alignements, et des Media Queries pour assurer un rendu fluide de la montre connectée jusqu'à l'écran de bureau.
